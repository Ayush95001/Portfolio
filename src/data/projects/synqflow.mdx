# SynqFlow: Event-Driven Automation Platform

## Overview

SynqFlow is a lightweight automation platform inspired by Zapier that allows users to connect triggers to actions and execute workflows automatically. It is designed around real-world backend patterns such as event streaming, asynchronous workers, and reliable execution.

## What Users Can Do

- **Create Users:** Sign up and manage user accounts.
- **Build Synqs:** Create automations by selecting webhook-based triggers and one or more actions.
- **Chain Actions:** Execute multi-step workflows asynchronously and sequentially.
- **Dynamic Data Passing:** Use template variables like `{user.email}` or `{order.amount}` between steps.
- **Track Executions:** Persist workflow execution metadata for visibility and reliability.
